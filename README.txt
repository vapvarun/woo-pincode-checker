== Description ==

Allows the site admin to add the pincode availability feature on their sites with woo pincode checker plugin. If you are starting your online shopping site, the Woo pincode checker is a perfect option for you.

WooCommerce Pincode checker is a vital feature to check the availability of a product according to the pincode. With this plugin, customers can know the timeline of your available orders before placing orders as this plugin can tell about the delivery dates of products.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Download the zip file and extract it.
2. Upload `woo-pincode-checker` directory to the `/wp-content/plugins/` directory
3. Activate the plugin through the \'Plugins\' menu.
4. Enjoy
